# 西安石油大学校园二手商品交易网功能一览

## 1. 用户管理系统

### 1.1 用户认证
- **用户注册**：学生通过学号、姓名、密码等信息注册账号
- **用户登录**：支持用户名密码登录
- **权限管理**：区分普通用户和管理员角色

### 1.2 用户信息管理
- **个人资料编辑**：修改头像、姓名、邮箱、电话、学院等信息
- **密码修改**：支持用户修改登录密码

## 2. 商品管理系统

### 2.1 商品发布
- **发布商品**：填写商品标题、描述、价格、原价、分类、新旧程度、交易方式等信息
- **图片上传**：支持多张商品图片上传

### 2.2 商品管理
- **商品编辑**：修改已发布商品的信息
- **商品下架**：将商品从在售状态转为下架状态
- **商品删除**：删除已发布的商品
- **我的商品**：查看自己发布的所有商品，支持按状态筛选

### 2.3 商品浏览
- **商品列表**：展示所有在售商品，支持按分类、价格、状态筛选
- **商品搜索**：支持关键词搜索商品
- **商品排序**：支持按最新发布、价格升序、价格降序、热度排序
- **商品详情**：查看商品详细信息、卖家信息、交易方式等

### 2.4 商品收藏
- **收藏商品**：将感兴趣的商品加入收藏夹
- **取消收藏**：从收藏夹中移除商品
- **收藏列表**：查看所有收藏的商品

### 2.5 商品分类
- **分类浏览**：按商品分类查看商品
- **分类管理**：管理员可新增、编辑、删除商品分类

## 3. 订单管理系统

### 3.1 订单创建
- **购买商品**：选择商品数量和交易方式，创建订单
- **订单确认**：确认订单信息和收货地址

### 3.2 订单状态管理
- **待支付**：创建订单后等待支付
- **待发货**：买家支付后，卖家准备发货
- **待收货**：卖家发货后，等待买家确认收货
- **已完成**：买家确认收货，交易完成
- **已取消**：取消未完成的订单

### 3.3 订单操作
- **取消订单**：买家或卖家可取消符合条件的订单
- **确认收货**：买家确认收到商品
- **发货确认**：卖家确认发货或面交完成

### 3.4 订单查询
- **买家订单**：查看买家视角的所有订单
- **卖家订单**：查看卖家视角的所有订单
- **订单详情**：查看订单的详细信息和交易进度

### 3.5 订单评价
- **评价商品**：交易完成后，买家对商品进行评价
- **评价管理**：查看已评价的订单

## 4. 管理员系统

### 4.1 仪表盘
- **数据统计**：查看平台商品、订单、用户数量统计
- **销售趋势**：展示销售数据趋势图

### 4.2 用户管理
- **用户列表**：查看所有注册用户
- **用户详情**：查看用户详细信息
- **用户状态管理**：冻结/解冻用户账号

### 4.3 商品管理
- **商品审核**：审核用户发布的商品，决定是否上架
- **商品列表**：查看所有商品，支持按状态筛选
- **商品详情**：查看商品详细信息

### 4.4 订单管理
- **订单列表**：查看所有订单
- **订单详情**：查看订单详细信息
- **订单状态管理**：处理异常订单

### 4.5 分类管理
- **分类列表**：查看所有商品分类
- **分类编辑**：新增、编辑、删除商品分类

### 4.6 统计分析
- **商品统计**：商品数量、分类分布统计
- **订单统计**：订单数量、金额统计
- **用户统计**：用户增长趋势统计

## 5. 交易方式

- **面交**：买卖双方约定线下地点交易
- **邮寄**：卖家通过快递发货，买家收货后确认
- **均可**：买卖双方可协商选择交易方式

## 6. 其他功能

- **文件上传**：支持商品图片上传
- **响应式设计**：适配不同设备屏幕
- **交易安全**：提供评价系统保障交易安全

# 系统架构

## 前端技术栈
- **框架**：Vue 3 + TypeScript
- **构建工具**：Vite
- **UI组件**：未明确使用特定组件库
- **状态管理**：Pinia
- **路由**：Vue Router
- **HTTP客户端**：Axios

## 后端技术栈
- **框架**：Spring Boot
- **数据库**：MySQL
- **ORM**：MyBatis
- **认证**：JWT
- **构建工具**：Maven

## 部署环境
- **前端**：Nginx
- **后端**：Java 8+
- **数据库**：MySQL 5.7+

# 系统特点

1. **面向校园**：专门为西安石油大学学生设计，支持学号认证
2. **交易安全**：提供评价系统和交易流程保障
3. **操作简单**：界面友好，操作流程清晰
4. **功能全面**：涵盖商品发布、浏览、交易、管理等全流程
5. **响应式设计**：适配PC端和移动端
6. **数据统计**：提供多维度的数据统计和分析
